# visit-bavaria
## Description

A website dedicated to exploring the rich history of Bavaria.  
The project provides reliable, evidence-based information about artefacts, people, places, and events, supported by timelines, images, and videos.  
It is designed with accessibility, user experience, and responsivity at its core.

---

## ✨ Features
- Interactive timeline of historical events
- Evidence-based articles with references
- Images and videos supporting the facts
- Structured navigation menu
- Accessible design (alt text, color contrast, semantic markup)
- Responsive layout using CSS media queries
- User-controlled audio/video playback

---

## 📐 Design Principles
- Clear navigation and structured layout
- Accessibility compliance (contrast, alt text, semantic HTML)
- Information organized by priority with headers and categories
- Consistent graphics and color scheme
- Foreground content never distracted by backgrounds

---

## 🧪 Testing & Validation
- Minimum of three pages (or three scrollable sections)
- Custom HTML passes W3C validation
- Custom CSS passes Jigsaw validation
- Images optimized for resolution and clarity
- External links open in new tabs
- Responsive design tested across devices
- Semantic markup used throughout
- Site-specific content (no placeholder text)

---

## ☁️ Deployment
- Hosted on GitHub Pages
- Version control managed via Git & GitHub
- Final deployment matches development version
- No broken internal links

## Sections of the website
### Homepage
Introduces visitors to Bavaria with a hero image, welcome text, and quick navigation links.

### Themes
Explores overarching cultural and historical themes that define Bavarian identity.

### Biographies
Profiles influential figures from Bavarian history, highlighting their contributions and legacy.

### Timelines
Interactive chronological view of key events, allowing users to explore history dynamically.

### Places
Showcases landmarks, castles, and significant locations across Bavaria.

### Events
Covers traditional festivals, cultural gatherings, and historical milestones.

### Resources
Provides references, external links, and supporting materials for deeper exploration.

### About
Explains the purpose of the project, its creators, and guiding principles.

### Contact
Offers ways to reach out for feedback, collaboration, or inquiries.

### Sign-up
Allows users to register for updates, newsletters, or participation in community features.
